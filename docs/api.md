# API Reference

## ember.Tensor

The core class of the library.

### Methods

- `__init__(size: int)`
- `to_list() -> list`: Convert tensor data to a Python list.
- `to_np() -> numpy.ndarray`: Convert tensor data to a NumPy array.

## Functional API

- `ember.add(a, b)`
- `ember.sub(a, b)`
- `ember.mul(a, b)`
- `ember.div(a, b)`
- `ember.sin(x)`
- `ember.cos(x)`
- ... and more.
